## Color Reference

### Status Colors
- **Answered Questions**: `#d1fae5` (light green) with `#10b981` (green) border
- **Unanswered Questions**: `#fee2e2` (light red) with `#ef4444` (red) border
- **Marked for Review**: `#fef3c7` (light yellow) with `#f59e0b` (orange) border
- **Current Question**: `#ede9fe` (light purple) with `#667eea` (purple) border

### Button Colors
- **Primary (Next, Submit)**: `#667eea` to `#764ba2` gradient
- **Secondary (Previous, Cancel)**: White with `#d1d5db` border
- **Mark for Review**: `#fbbf24` (yellow) border
- **Success**: `#10b981` (green)

### Progress Bar
- **Background**: `#e5e7eb` (light gray)
- **Fill**: `#10b981` to `#059669` gradient (green)

## Animation Notes

### Transitions
- Modal appears: 0.3s fade in
- Content slides up: 0.4s ease
- Progress bar fills: 0.4s ease
- Button hover: 0.2s ease
- Answer selection: Instant highlight

### Micro-interactions
- Button hover: Slight lift (translateY -2px) + shadow
- Option hover: Border color change + background tint
- Auto-save indicator: Slide in from right, fade out after 2s
- Timer warning: Pulse animation at 5 minutes

---

## Responsive Behavior

### Desktop (1000px+)
- Modal width: 1000px max
- Two-column grid for review (5x2)
- Full navigation bar visible

### Tablet (768px - 999px)
- Modal width: 95% viewport
- Two-column grid for review (5x2)
- Condensed header (stacked)

### Mobile (< 768px)
- Modal: Full screen (100vw x 100vh)
- Single-column grid for review
- Header items stack vertically
- Footer buttons wrap to multiple rows
- True/False buttons stack vertically

│                                                         │
